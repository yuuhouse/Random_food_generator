

# 隨機吃啥產生器

突發奇想臨時用 GPT-4o 復刻了我以前用 C# 寫的隨機菜單產生器。

![image](https://github.com/yuuhouse/Random_food_generator/blob/main/Screenshot%202024-05-29%20220427.png)

## 目的
- 不知道吃啥的時候就按按鈕。

## 使用方式
1. 下載ZIP或Clone此Repositories。
2. 使用合適的開發工具打開項目。
3. 運行py程式以生成隨機菜單。
4. 可以自己想辦法另外包成exe。
